## Overview
ChainiForge is a "One-Click" deployment platform for Polygon CDK Layer 2 networks. It simplifies the complex process of infrastructure setup, compliance integration, and economic configuration into a streamlined web interface.

## Tech Stack
- **Frontend**: Vite + React + Vanilla CSS (Premium design with glassmorphism)
- **Icons**: Lucide React
- **Animations**: Framer Motion (for smooth transitions)
- **Backend Simulation**: Node.js (to simulate infrastructure orchestration)
- **Smart Contracts**: Solidity (for Paymaster and Verified Bridge templates)
